# Viktor Seleznev — Tour Guide Portfolio

Статичный одностраничный сайт-резюме для профессионального гида во Вьетнаме.

## Идея проекта

Интерактивное резюме на двух языках (RU/EN), которое можно отправить работодателю вместо PDF. Вместо скучного документа — живой сайт с фотографиями туристов, отзывами и опытом работы.

## Структура

```
index.html       — весь сайт в одном файле (HTML + CSS + JS)
hero-photo.jpg   — фото гида для главного экрана
hero-bg.jpg      — фоновое фото для hero-секции
туристы/         — фотографии с туристами (слайдшоу)
Я/               — личные фото гида (слайдшоу)
места/           — фотографии локаций (сетка 3×3)
```

## Секции сайта

1. **Hero** — имя, должность, краткое описание, контакты
2. **О себе** — биография и ключевые компетенции
3. **Туристы** — слайдшоу с групповыми фотографиями
4. **Опыт работы** — карточки с местами работы
5. **Гид в деле** — личные фото на экскурсиях
6. **Отзывы** — карусель с реальными отзывами с Google Maps
7. **Локации** — сетка мест, которые показывает гид
8. **Контакты** — Telegram и номер телефона

## Технологии

- Чистый HTML/CSS/JS, без фреймворков
- Адаптивный дизайн (мобиль + десктоп)
- Переключение языка RU/EN без перезагрузки
- Слайдшоу с кроссфейдом и свайпом
- Деплой на Vercel: [viktor-seleznev-portfolio.vercel.app](https://viktor-seleznev-portfolio.vercel.app)

## Как обновить фото

Заменить файл в соответствующей папке (`туристы/`, `Я/`, `места/`) с тем же именем, сделать `git push` — Vercel перезальёт автоматически.
